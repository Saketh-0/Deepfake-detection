# Deepfake Detection

A web application built in **Python (Flask)** for detecting deepfake videos using a trained machine learning model.  
This project lets users upload videos and get predictions on whether the content is **real** or **fake**.

---

##  Features

- Upload videos through a simple web UI  
- Flask backend for processing and serving results  
- Machine learning model for deepfake classification  
- Organized directory structure for easy maintenance  
- Displays the probability score and prediction result  

---

## Run the Application

Start the Flask server:
python app.py

## Example Input
Upload a video file

## Example Output
Deepfake probability: 0.87
Prediction: Fake
